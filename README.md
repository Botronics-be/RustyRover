# Introduction

:wave: _Hello to our RustyRover diff drive robot repository !!_

This repo aims to show an example of a simple diff drive rover using [Pixi](https://pixi.prefix.dev/latest/) and the [rclrs](https://github.com/ros2-rust/ros2_rust) ROS2 client library.
